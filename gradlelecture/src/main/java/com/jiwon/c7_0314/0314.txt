상속의 장점
    - 코드를 다시 개발하지 않아도 된다.(재사용)
        계산기 + 추가 기능
    - if else 를 안써도 된다.
        부모타입으로 만들어야 한다.

단점
    - 단일 상속 밖에 할 수 없다.


추상 클래스
    - 추상클래스는 직접적이 new를 할 수 없다.

추상 메소드
    - 일부 물려주고 + 재정의 할떄 강제성을 준다
    - if else 패턴
